## SEO (Search Engine Optimization)

SEO is the practice of optimizing web pages so search engines can accurately understand, index, and rank them. It improves visibility by enhancing metadata, structure, performance, accessibility, and keyword relevance.

## OGP (Open Graph Protocol)

OGP defines how a webpage appears when shared on social platforms. It uses structured metadata to control the preview card’s title, description, image, and URL across networks like Facebook, LinkedIn, WhatsApp, and Discord.

## Semantic Web Layout

Semantic layout uses meaningful HTML elements such as `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, and `<footer>`. These elements give structure and context to content, improve accessibility, and help search engines understand page hierarchy.

## Twitter Cards

Twitter Cards provide metadata that defines how links appear on Twitter/X. They allow custom titles, descriptions, and large preview images, giving more control over link presentation and improving engagement.

## Canonical URL

A canonical URL tells search engines which version of a page is the primary source. It prevents duplicate content issues and consolidates ranking signals to the correct URL.

## Meta Description

A meta description is a brief summary of a page’s content. It appears in search results and influences click-through rates by giving users context about the page.

## Favicon

A favicon is the small icon displayed in browser tabs, bookmarks, and shortcuts. It enhances branding and improves the professional appearance of a website.

## Responsive Design

Responsive design ensures that a website adapts to different screen sizes using fluid layouts, flexible images, and CSS breakpoints, providing a consistent experience on all devices.
